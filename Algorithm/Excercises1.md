# Taller en clase – Introducción al pseudocódigo

**Curso: Fundamentos de Programación**
**Carácter:** No calificable
**Modalidad:** Trabajo individual o en parejas
**Duración sugerida:** 60–90 minutos

---

## Objetivo del taller

El objetivo de este taller es que el estudiante practique:

* La lectura y comprensión de problemas redactados.
* La identificación de variables.
* La construcción del bloque **VARIABLES**.
* La escritura de **pseudocódigo** usando:

  * Estructura secuencial
  * Estructura selectiva (SI / SINO)
* El uso de notación matemática y lógica.

---

## Indicaciones generales

Antes de escribir el pseudocódigo, **en todos los ejercicios** el estudiante debe:

1. Leer cuidadosamente el problema.
2. Identificar las variables necesarias.
3. Definir el bloque **VARIABLES**.
4. Escribir el pseudocódigo completo.

### Convenciones obligatorias

* Usar pseudocódigo, **no lenguajes de programación**.
* Usar bloques `text` para el pseudocódigo.
* El bloque **VARIABLES debe ir antes del INICIO**.
* Usar únicamente:

  * LEER
  * ESCRIBIR
  * SI / ENTONCES / SINO / FIN SI
* No usar estructuras iterativas.
* Usar notación matemática y lógica estándar (≥, ≤, ≠).

## Ejercicio 1 – Cálculo de promedio

**Problema:**
Un estudiante ingresa tres notas.
El sistema debe calcular el promedio y mostrarlo.

**Actividad:**

* Identifique las variables.
* Escriba el pseudocódigo que resuelve el problema.

## Ejercicio 2 – Mayor de edad

**Problema:**
Una persona ingresa su edad.
Si la edad es mayor o igual a 18, el sistema debe indicar que es mayor de edad.
En caso contrario, debe indicar que es menor de edad.

**Actividad:**

* Identifique las variables.
* Escriba el pseudocódigo correspondiente.

---

## Ejercicio 3 – Total a pagar

**Problema:**
Un cliente ingresa el valor de una compra.
Si el valor es mayor a 200 000, se aplica un descuento del 10 %.
En caso contrario, el valor a pagar es el mismo de la compra.

**Actividad:**

* Identifique las variables.
* Escriba el pseudocódigo que calcule y muestre el total a pagar.
